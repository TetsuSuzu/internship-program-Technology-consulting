## クイズ（理解確認の例）

> **Q. GitHub Codespace のライフサイクルにはどのような段階がありますか？**
> 1. Create, Rebuild, Stop, Delete ← ✅ 正解
> 2. Create, Delete
> 3. Create, Stop, Delete
> 4. Create, Rebuild, Delete

各セクション末にこの形式の3択クイズを置き、理解を確認する。

## GitHub Pages を使わない理由

企業のGitHub環境では public リポジトリが禁止 → private になり、private での Pages は有料プランが必要。
→ **公開は Codespacesプレビュー / README.md ポータル / Issues で代替**（詳細 `docs/06`）。
