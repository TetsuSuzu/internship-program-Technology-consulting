# 🚀 スキルアップ・ポータル（ハンズオン教材）

DXC インターンシップ ハンズオン用のテンプレートです。
このリポジトリを **Codespaces** で開いて、自分だけの「スキルアップ・ポータル」を作ります。

---

## 📁 ファイルの中身

| ファイル | 役割 | 書きかえる？ |
|----------|------|:---:|
| `index.html` | ページの中身（文章・見出し） | ✅ たくさん書きかえる |
| `style.css`  | 見た目（色・デザイン）       | ◯ 色だけでもOK |
| `README.md`  | この説明書                   | あとで |

---

## ✏️ やること（ステップ）

- [ ] Codespace を起動する
- [ ] `index.html` の「★ここを書きかえよう」を自分の内容にする
- [ ] `style.css` の色を好きな色に変える
- [ ] プレビューで表示を確認する
- [ ] 変更を保存（コミット）する

> 詳しい手順は **[Codespacesプレビュー手順書](./Codespacesプレビュー手順書.md)** を見てね。

---

## 💡 ヒント

- うまくいかない時は、ペアの相手と画面を見せ合おう
- 「★ここを書きかえよう」とコメントがある場所を探すと迷わないよ
