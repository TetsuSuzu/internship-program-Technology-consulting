# 🖥️ Codespaces プレビュー手順書

> **このハンズオンのゴール**
> 自分のPCに何もインストールせず、ブラウザだけで Web ページを編集して、その場で表示（プレビュー）できるようになる！

所要時間：約 20〜30 分 ／ 対象：はじめての人

---

## ✅ はじめる前に（準備）

- [ ] GitHub アカウントにサインインできている
- [ ] 使うリポジトリを開ける（講師が共有したリンク）

> 💬 困ったら、まわりのメンバーや講師にすぐ聞いてOKです！

---

## STEP 0　GitHub にサインインする

1. ブラウザで [https://github.com](https://github.com) を開く
2. 右上の **「Sign in」** からサインインする

> 📸 **【スクショ①：GitHubのサインイン画面】**
> （ここにサインイン画面の画像を貼る）

---

## STEP 1　Codespace を起動する

1. 講師が共有した **リポジトリのページ**を開く
2. 緑色の **「< > Code」** ボタンをクリック
3. **「Codespaces」** タブを選ぶ
4. **「Create codespace on main」** をクリック

> 📸 **【スクショ②：Code ボタン → Codespaces タブ】**
> （緑のCodeボタンを押してCodespacesタブが開いた画面）

5. 画面が切り替わり、**準備中（Setting up your codespace）** と表示される
6. 30秒〜1分ほど待つと、ブラウザの中に **編集画面（VS Code）** が開く 🎉

> 📸 **【スクショ③：Codespace（VS Code）が開いた画面】**
> （左にファイル一覧、中央にエディタが見える状態）

> 🧩 **豆知識：Codespace のライフサイクル**
> Codespace には `Create（作成）→ Rebuild（再構築）→ Stop（停止）→ Delete（削除）` という段階があります。
> 今あなたがやったのは **Create** です。使い終わったら最後に **Stop** します（STEP 5）。

---

## STEP 2　ファイルを開いて書きかえる

1. 左側の **ファイル一覧（エクスプローラー）** から **`index.html`** をクリック
2. ファイルの中の **「★ここを書きかえよう」** を探す
3. 自分の名前や自己紹介に書きかえる

> 📸 **【スクショ④：index.html を開いて編集している画面】**
> （「★ここを書きかえよう」のコメントが見える状態）

4. 書きかえたら、保存する
   - **Windows：`Ctrl` + `S`** ／ **Mac：`⌘` + `S`**

> 💡 色を変えたい人は `style.css` を開いて、`--main-color` の値を変えてみよう！

---

## STEP 3　簡易サーバを立てる（ターミナル）

1. 上のメニューから **「Terminal（ターミナル）」→「New Terminal」** を選ぶ
   - ショートカット： **`Ctrl` + `@`**（バッククォートキー）

> 📸 **【スクショ⑤：ターミナルの開き方】**

2. 画面の下に黒い入力欄（ターミナル）が出る
3. まず `index.html` がある **handson フォルダに移動**して、サーバを立てます。
   次の2行を1行ずつ入力して **Enter** を押す：

   ```bash
   cd handson
   python3 -m http.server 8000
   ```

   > 💡 移動せずに1コマンドで済ませたい場合：
   > ```bash
   > python3 -m http.server 8000 --directory handson
   > ```

4. `Serving HTTP on ... port 8000 ...` と表示されたら成功！

> 📸 **【スクショ⑥：サーバ起動に成功したターミナル】**

---

## STEP 4　プレビューを表示する 🎉

1. 右下に **「Open in Browser（ブラウザで開く）」** という通知が出る → クリック

> 📸 **【スクショ⑦：右下のOpen in Browser通知】**

2. 通知が出ない場合は…
   - 画面下の **「PORTS（ポート）」タブ** をクリック
   - **`8000`** の行にある 🌐 **地球アイコン（Open in Browser）** をクリック

> 📸 **【スクショ⑧：PORTSタブと地球アイコン】**

3. 新しいタブで **自分のページが表示されます！** 🎊

> 📸 **【スクショ⑨：完成したページのプレビュー】**

---

### 🔁 直したら、もう一度見るには？
1. `index.html` を直して **保存（Ctrl+S）**
2. プレビューのタブを開いて **更新（リロード / F5）**
3. → 変更が反映されます

---

## STEP 5　終わったら Codespace を「停止」する

使い終わったらそのままにせず、必ず止めましょう（無駄な消費を防ぎます）。

1. 左下の **「Codespaces」** または画面右上のメニューを開く
2. **「Stop Current Codespace（停止）」** を選ぶ

> 📸 **【スクショ⑩：Stop Current Codespace のメニュー】**

> 🧩 これがライフサイクルの **Stop** です。次に開くときは続きから再開できます。

---

## 🆘 こまったときは（トラブル対処）

| 症状 | 対処 |
|------|------|
| ページが真っ白 | `index.html` を保存したか確認 → プレビューを更新（F5） |
| `python3` が無いと出る | `python -m http.server 8000` で試す |
| プレビューが開かない | 下の「PORTS」タブから 8000 の🌐をクリック |
| ターミナルが消えた | メニュー Terminal → New Terminal でもう一度開く |
| ぜんぶ分からない | 画面を見せて講師・メンバーに聞く！（これが一番大事） |
